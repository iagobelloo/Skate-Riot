# Skate-Riot

# Sistema de Gestão com Access e VBA 

Projeto prático desenvolvido para treinar lógica de programação, manipulação de banco de dados e automação de processos.

A ideia foi criar uma aplicação desktop funcional utilizando o ecossistema Microsoft para resolver problemas de cadastro e consulta de dados.

## Tecnologias
•⁠  ⁠*Microsoft Access:* Banco de dados relacional.
•⁠  ⁠*VBA (Visual Basic for Applications):* Back-end, automação e validações.
•⁠  ⁠*SQL:* Consultas internas.

## Funcionalidades
O sistema conta com as principais operações de um CRUD:
•⁠  ⁠*Formulários:* Interface amigável para input de dados.
•⁠  ⁠*Validações:* Scripts em VBA para garantir que os dados inseridos (como CPF ou valores) estejam corretos antes de salvar.
•⁠  ⁠*Automação:* Botões e macros para limpar campos, gerar relatórios e navegar entre telas.
•⁠  ⁠*Relatórios:* Geração de views automáticas para análise rápida.

## Aprendizado
Foquei bastante em entender como o código interage com os objetos do banco. Trabalhei com depuração (debugging) para corrigir erros de lógica e melhorei a estrutura dos formulários para facilitar o uso final.
